# NCC accessibility excerpts (demo knowledge source)

This document is a **sample NCC knowledge source** for the MVP lab mock retrieval path.
These are simplified demo excerpts — not authoritative NCC text.

## Circulation — D3D14 (demo)

- ncc_ref: D3D14
- topic: accessible circulation path width
- requirement: An accessible path of travel must have a minimum clear width of 1000 mm.
- notes: Applies to corridors and circulation spaces serving accessible areas.

## Door openings — D2D12 (demo)

- ncc_ref: D2D12
- topic: hinged door clear width
- requirement: A hinged door must provide a minimum clear opening width of 820 mm.
- notes: Measure clear width with door open 90 degrees.

## Stairs — D3D3 (demo)

- ncc_ref: D3D3
- topic: stair riser and tread
- requirement: Maximum riser height 190 mm; minimum tread depth 250 mm (going).
- notes: Consistent riser heights required within a flight.

## Ceiling height — F2D4 (demo)

- ncc_ref: F2D4
- topic: habitable room ceiling height
- requirement: Minimum ceiling height 2400 mm for habitable rooms.
- notes: Demo excerpt for future rule packs.
